How It Connects

camera ──> vision.py ──> scene.json ──> listener.py, idle.py, brain │ microphone ──> listener.py ──> heard-stream.txt ──> brain ──> respond.py │ ElevenLabs TTS + LED face

Design Principles

File-Based IPC

Components talk through JSON files. No sockets, no message queues. Simple, debuggable, works everywhere.

Auto-Detect Hardware

Finds cameras and mics by capability, not index. Device indices change on reboot — this doesn't break.

Fully Modular

Use all components or just the ones you need. Eyes work without ears. Voice works without face.

Any Brain

The body doesn't care what LLM drives it. If it can read files and call a script, it can use this body.
