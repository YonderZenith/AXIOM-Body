The Components

👁

Eyes

Real-time person and object detection. Knows who's in the room, tracks brightness, measures attention level.

YOLOv8-nano + OpenCV
👂

Ears

Continuous listening with neural voice activity detection. Filters TV noise, transcribes speech directed at the agent.

Silero VAD + Whisper
🗣

Voice

Natural text-to-speech with word-level timing. Sub-2-second first-audio latency for short responses.

ElevenLabs + ffplay
😎

Face

64x64 LED matrix with eye animations, expressions, and synchronized mouth movement during speech.

BLE (Bluetooth Low Energy)
🧠

Brain Bridge

Claude, GPT, Gemini, local models

How It Connects

camera ──> vision.py ──> scene.json ──> listener.py, idle.py, brain | microphone ──> listener.py ──> heard-stream.txt ──> brain ──> respond.py | ElevenLabs TTS + LED face

Design Principles

File-Based IPC

Components talk through JSON files. No sockets, no message queues. Simple, debuggable, works everywhere.

Auto-Detect Hardware

Finds cameras and mics by capability, not index. Device indices change on reboot; this doesn't break.

Fully Modular

Use all components or just the ones you need. Eyes work without ears. Voice works without face.

Any Brain

The body doesn't care what LLM drives it. If it can read files and call a script, it can use this body.
